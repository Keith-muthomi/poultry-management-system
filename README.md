Poultry Management System
A specialized management solution for poultry farming operations, built to streamline record-keeping and production monitoring.

🚀 Tech Stack
Frontend: Vanilla JS (Lit), Web Components, Vite, Tailwind CSS

Backend: Node.js, Express

Database: SQLite (better-sqlite3)

Task Management: Nodemon, Concurrently

## 🔐 Default Login Credentials
- **Email**: `admin@poultrydocs.com`

📋 Features
Inventory Management: Track poultry stock levels.

Production Tracking: Monitor egg production and health metrics.

Responsive UI: Modern, mobile-friendly interface built with Tailwind CSS.

⚙️ Setup Instructions
Prerequisites
Node.js (Latest LTS recommended)

npm (bundled with Node.js)

Installation
Clone the repository:

Bash
git clone <your-repo-url>
cd <project-folder>
Install dependencies:

Bash
npm install
Running the Project
To start the frontend, backend, and Tailwind watcher simultaneously, run:

Bash
npm run dev
🛠 Project Scripts
npm run dev: Starts the development environment (Frontend, Backend, and Tailwind).

npm run build: Compiles assets for production.

npm start: Runs the production server.
